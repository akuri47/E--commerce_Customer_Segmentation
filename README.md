
# E-commerce Customer Segmentation & Recommendation System

This project analyzes customer purchasing behavior using transaction data and builds a recommendation system using market basket analysis.

## Features

- Exploratory Data Analysis (EDA)
- RFM Feature Engineering
- Customer Segmentation using:
  - K-Means
  - DBSCAN
  - Hierarchical Clustering
- Market Basket Analysis using FP-growth
- Product Recommendation System

## Tech Stack

- Python
- Pandas
- NumPy
- Scikit-Learn
- MLxtend
- Matplotlib
- Seaborn

## Dataset

Online Retail II Dataset

## Results

- Identified 5 distinct customer segments
- Generated association rules for product recommendations
- Built a recommendation system based on purchasing patterns
